AWS Lambda Server for SudoRPC

```sh
yarn add @sudorpc/server-aws-lambda
yarn add @sudorpc/core # Peer Dependency
# Or
npm install @sudorpc/server-aws-lambda --save
npm install @sudorpc/core --save # Peer Dependency
```
